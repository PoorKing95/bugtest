#正则，用来匹配一类具有相同规则的字符串
#使用功能:1.正则匹配  2.正则替换
#单字符：
# . :除换行以为所有字符
# []:[aoe]:三个里边任意一个匹配。 [a-w]:匹配集合中任意一个字符
# \d:数字0-9 
# \D：非数字
# \w：数字、字母、下划线 
# \W：非\w 
# \s：所有空白字符
# \S：所有非空白字符 
# 数量修饰词 ：
# *:任意多次，大于等于0次
# +：大于等于一次 
# ?：可有可无 0或1次
# {m}：固定m次 
# {m,}：至少n次
# {m,n}：m到n次
# 边界：
# $:以某某结尾
# ^：以某某开头
# 分组：
# (){4}:视为一个整体，（）中的内容出现4次
# ()子模式 \1 \2
# 贪婪模式：
# .*? 与 .+?
# re.I : 忽略大小写
# re.M : 多行匹配
# re.S : 不加re.S，内容的每一次换行都会从头开始，加上re.S内容不会因为换行而重头开始匹配。
